## Description
### This is the repository for a mock back-end of an e-commerce business that allows users to create, read, update, and delete inventory information from command line using the sequelize package and mysql.
## Installation
### Download the repository, navigate to the root file in the terminal of your choice, then use 'npm i' to download dev dependencies.
## Usage
### Use the server.js file in the root folder to set up the Sequelize connection to the database.
